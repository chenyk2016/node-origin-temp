# node项目从0-1

## 创建项目

```py

# 初始化git仓库，添加.gitignore文件
git init

# 初始化node项目
npm init
```

## 配置eslint

```zsh
npm install eslint --save-dev

./node_modules/.bin/eslint --init
```

```json

```

